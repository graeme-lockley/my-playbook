# az bicep

[az bicep](https://docs.microsoft.com/en-us/cli/azure/bicep) is a plugin to [Azure CLI](https://docs.microsoft.com/en-us/cli/azure).

## Installation

| Device | Process |
|-|-|
| Mac dotfiles |  module `azure-tools` |
| Mac adhoc | `brew install azure-cli` |
